# 📝 To-Do API на Go

Простой REST API для управления задачами, реализованный на языке Go. Поддерживает регистрацию пользователей, вход по логину и авторизацию с помощью JWT-токенов. Пользователь может создавать, просматривать и управлять своими задачами.

---

## 🚀 Возможности

- ✅ Регистрация пользователей
- 🔐 Авторизация с JWT
- 📋 CRUD-операции для задач
- 🧾 Защищённый доступ через middleware
- 💾 PostgreSQL для хранения данных

---

## ⚙️ Технологии

- Go (Golang)
- PostgreSQL
- Gorilla Mux (роутер)
- JWT (golang-jwt)
- bcrypt (шифрование паролей)
- Postman (тестирование API)
